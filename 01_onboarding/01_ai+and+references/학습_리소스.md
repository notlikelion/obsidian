### 자소서 작성 프롬프트 예시

```
Persona : 너는 10년차 클라우드 엔지니어링 전문가이자 면접관이야
Task : 주니어를 위한 합격 자소서 예시를 작성해줘.
Context : AWS, Java Spring Boot, MySQL 등을 스펙으로 해.
Format : 3가지 버전으로 각각 1000자씩 작성해줘.
```
