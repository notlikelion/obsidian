## Mac vs Windows 환경 차이

### 설정 창 호출

- **macOS**: `⌘,` (IntelliJ IDEA › Preferences)
- **Windows**: `Ctrl + Alt + S` (File › Settings)

### 키보드 단축키

- **macOS**: `⌘`·`⌥` 기반
- **Windows**: `Ctrl`·`Alt` 기반
- **변경 방법**: Settings › Keymap에서 교차 변경 가능

### 디렉터리 구분자

- **Mac/Linux**: `/`
- **Windows**: `\`
