## 프로젝트 생성 및 초기 설정

### 언어 설정

- 한글이 아닌 경우: **Customize** → 언어 설정에서 한국어 선택

### 첫 프로젝트 생성

1. Welcome 화면 → **New Project** 클릭
2. **Build System 선택**: IntelliJ
3. **JDK 17** 지정 (없으면 Download → Zulu 선택)
4. **프로젝트 위치**: 본인이 실습하는데 쓸 폴더 지정
5. **Finish** 클릭

### 기본 설정 복원

- **파일 > IDE 설정 관리 > 디폴트 설정 복원**
